# Material Specifications

When generating teaching materials via the `create_material` tool, follow these format specifications for each material type.

## Word Bank

- Two-column format: term and student-friendly definition
- Include 8-15 key terms from the activity
- Definitions should be grade-appropriate (match the teacher's grade range)
- If bilingual support is a classroom need, add a third column for translations or cognates
- Terms should be sorted by order of appearance in the activity, not alphabetically

## Graphic Organizer

- Choose the organizer type based on the activity's cognitive demand:
  - **Compare/contrast** → Venn diagram or T-chart
  - **Sequence/process** → Flow chart or timeline
  - **Cause/effect** → Two-column with arrows
  - **Main idea/details** → Web or tree diagram
  - **Categorization** → Sorting chart with labeled columns
- Include clear headings and example entries (1-2 filled in as models)
- Leave adequate blank space for student responses
- Use simple, clean formatting

## Rubric

- 3-4 criteria aligned with the activity's learning objectives
- 3 performance levels: Beginning, Developing, Proficient
- Use student-friendly language at the teacher's grade level
- Each cell should describe observable, concrete behaviors
- Include a self-assessment column if the classroom has check-in needs

## Simplified Instructions

- Rewrite activity instructions at 2 grade levels below the original
- Break multi-step instructions into numbered single steps
- Bold key action words (read, write, circle, discuss)
- Add visual cues where helpful (icons or simple descriptions of icons)
- Preserve all essential information — do not remove steps

## Choice Board

- 3x3 grid (9 options) or 2x3 grid (6 options) depending on activity scope
- Each cell describes one way to demonstrate learning
- Include options across UDL expression categories: written, verbal, visual, hands-on
- Mark the center square as "free choice" or "teacher's choice"
- Each option should be completable within the activity's time frame

## Sentence Starters

- 6-10 sentence frames that scaffold the activity's thinking/writing task
- Graduated complexity: start with simpler frames, progress to more complex
- Include transition words and academic vocabulary from the activity
- Group by purpose: describing, comparing, arguing, summarizing, etc.

## Vocabulary Chart

- Three-column format: word, definition, visual/sketch space
- Include pronunciation guide for complex terms
- Add a "use it in a sentence" row for each term
- If bilingual support is needed, add cognate/translation column
- 6-12 terms maximum per chart

## Checklist

- Task-completion checklist for the activity's steps
- Numbered with checkboxes
- Include "I need help" checkpoint markers at key decision points
- Add estimated time per step if the activity is time-bound
- End with a self-reflection question: "What was the hardest part?"
