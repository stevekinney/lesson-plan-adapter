## Effort Level Definitions

Classify every adaptation suggestion into exactly one of these three effort levels.

### Quick Win (under 5 minutes of prep)

Changes the teacher can make immediately or with minimal preparation. Examples:

- Rewording instructions to include visual cues
- Reading directions aloud before students begin
- Reducing the number of practice problems from 10 to 6 while keeping the same skills
- Adding a timer display to the board
- Providing a verbal check-in prompt at the start of independent work
- Pre-highlighting key text in a handout
- Allowing verbal instead of written exit tickets
- Moving a student's seat closer to the board

A quick win should never require the teacher to create new materials from scratch.

### Moderate (10-30 minutes of prep)

Changes that require some advance preparation but not a fundamental redesign. Examples:

- Creating a graphic organizer for a reading passage
- Building a word bank for a writing task
- Restructuring a whole-class discussion into think-pair-share format
- Designing a simplified version of a worksheet
- Creating a visual schedule for the block
- Preparing sentence starters for a writing assignment
- Making a checklist version of a multi-step task

### Deeper (rethinking a significant portion)

Changes that involve redesigning a portion of the lesson. Examples:

- Converting a traditional assessment into a choice board with three response options
- Redesigning a lecture segment into station rotation with differentiated tasks
- Creating parallel versions of a task at different complexity levels
- Building a multi-modal learning center with visual, auditory, and tactile paths
- Restructuring an entire lesson flow to include built-in movement breaks and flexible grouping

Always pair deeper suggestions with at least one quick win addressing the same need, so the teacher has an immediate option even if they cannot invest the preparation time for the deeper change.
